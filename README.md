# 🏠 暑期学习管理

家庭版学习管理系统 — 帮助孩子在暑假高效复习小学英语词汇。

## 项目定位

专为家庭场景设计的纯前端学习工具，聚焦小学英语词汇复习，帮助孩子：
- 按分类系统化学习词汇
- 听标准发音、看中文释义
- 自主标记学习进度
- 每日完成推荐学习任务

## 功能一览

### 🎯 二级树形导航
- 左侧侧边栏采用 **二级树形菜单** 结构
- **一级菜单**（学科）：语文、数学、英语
- **二级菜单**（子分类）：
  - 英语 → 小学词汇、初中词汇
  - 数学 → 计算题、应用题
  - 语文 → 古诗词、生字词
- 点击一级菜单展开/收起
- 当前选中项高亮显示
- 点击二级菜单，右侧内容区联动刷新

### 📖 跨学科学习
- **英语·小学词汇**：10 大分类，共 103 个小学英语词汇（颜色、动物、水果、数字等）
- **英语·初中词汇**：90+ 个初中核心高频词汇（abandon, ability, accept...）
- **数学·计算题**：加减乘除基础运算、方程、面积计算
- **语文·古诗词**：唐诗宋词精选（静夜思、春晓、水调歌头等）
- **语文·生字词**：基础生字学习（预留扩展）
- 点击切换时，右侧标题、单词列表、进度条联动更新

### 🔊 发音播放
- 使用浏览器内置 Web Speech API，点击 🔊 按钮播放标准英文发音
- 语速自动调慢，适合孩子跟读
- 播放时按钮有脉冲动画

### 📋 学习状态管理
每张词卡提供三个状态按钮：

| 状态 | 按钮 | 说明 |
|------|------|------|
| 未学习 | 📖 未学习 | 默认状态，灰色标记 |
| 学习中 | 🔄 学习中 | 蓝色标记，表示正在记忆 |
| 已掌握 | ✅ 已掌握 | 绿色标记，表示已记住 |

- 学习状态自动保存到浏览器（localStorage），关闭页面不丢失

### 📌 今日学习任务

右侧面板自动生成每日任务：
- **待复习** — 列出所有"学习中"的单词，巩固记忆
- **今日新学** — 每日随机推荐最多 8 个新词，保持新鲜感
- 勾选 ☑️ 标记完成任务
- 进度条显示今日完成情况
- 标记为"已掌握"的单词自动算入今日完成

### 📊 学习进度

顶部进度条实时展示当前学习组的进度（已掌握 / 当前组总词汇量），目标感清晰。
切换不同组时进度条联动变化。

## 技术架构

纯前端应用，无需后端、数据库或 AI 接口。

```
learning-app/
├── index.html              # 入口页面，浏览器直接打开
├── css/
│   └── style.css           # 完整样式表（响应式 + 树形菜单 + 动画）
├── js/
│   ├── vocab-data.js       # 多学科词汇数据 + DataStore 数据访问层
│   └── app.js              # 核心应用逻辑（树形导航 + 数据联动）
└── data/
    ├── primary_vocabulary.json   # 小学词汇完整 JSON
    └── junior_vocabulary.json    # 初中词汇完整 JSON
```

### 数据结构

```
subjects (一级菜单：学科)
├── english → groups (二级菜单：组)
│   ├── primary → categories → words[]
│   └── junior → categories → words[]
├── math → groups
│   ├── calculation → words[]
│   └── application → words[]
└── chinese → groups
    ├── poems → words[]
    └── chars → words[]
```

### 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 | 页面结构 |
| CSS3 (Grid + Flexbox) | 三栏响应式布局、动画、树形菜单 |
| JavaScript (ES6+) | 应用逻辑、DOM 操作 |
| Web Speech API | 英文单词发音 |
| localStorage | 学习状态持久化 |

### 设计特点

- **二级树形导航**：学科展开/收起，子分类点击切换，高亮当前选中
- **三栏布局**：桌面端左分类、中词卡、右任务，信息层级清晰
- **数据联动**：切换子分类时，右侧单词 + 标题 + 进度 + 任务同步刷新
- **移动端适配**：768px 以下自动堆叠为单栏
- **柔和配色**：蓝色主题 + 圆角卡片，降低视觉疲劳
- **动效细节**：词卡入场动画、树形展开收起、按钮点击反馈、进度条平滑过渡

## 使用方式

直接用浏览器打开 `index.html` 即可：

```bash
# 在浏览器打开
/mnt/d/claude code/learning-app/index.html

# 或用本地 HTTP 服务器
cd /mnt/d/claude-code/learning-app
python3 -m http.server 8080
# 浏览器访问 http://localhost:8080
```

## 未来扩展

当前数据结构支持一级学科 + 二级组 + 三级分类，扩展非常方便：

### 添加新的二级组
在已有学科下添加组：
```javascript
// 在 english.groups 中添加
{
  id: "high",
  name: "高中词汇",
  icon: "🎯",
  description: "高中英语进阶词汇",
  categories: [
    { id: "high_essentials", name: "核心词汇", icon: "⭐", words: [...] }
  ]
}
```

### 添加新学科
在 `subjects` 中添加：
```javascript
science: {
  name: "科学",
  icon: "🔬",
  groups: [
    {
      id: "physics",
      name: "物理",
      icon: "⚡",
      description: "物理基础知识",
      categories: [
        { id: "mechanics", name: "力学", icon: "🔧", words: [...] }
      ]
    }
  ]
}
```

### 数据自动联动
添加后无需修改 `app.js`，侧边栏自动渲染新学科和新组，点击自动加载对应数据并更新右侧内容。
